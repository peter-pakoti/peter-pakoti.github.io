Explain control flow and loops using an example process from everyday life.

Let's start off by explaining what each one is used for. JavaScript by default runs our code from top to bottom, line by line, however we can change this with control flow. Control flow allows our program to make decisions about what code is executed and when.

Now lets touch on loops! Loops are used in JavaScript to perform repeated tasks based on a condition. Conditions typically return true or false . A loop will continue running until the defined condition returns false .

If we take a look at the flow chart on our right it shows us how control flow and loops are used. Let me explain.

Let's say if we had a lamp that isn't working. The first step is to check if the lamp is plugged in. If the lamp is plugged in then it will go to the next instruction, however, if it is not plugged in then we'll plug it in before moving on to the next step in this case your code will repeat until it the first instruction is completed. Now our next instruction is to check if the light bulbs burned out, if it isn't then we know we need to repair the lamp and if it is then we know we need to change the bulb. In conclusion this is a very basic way of explaing work flow and looping and what they do.

Explain what functions are and why they are helpful.

A function is simply a “chunk” of code that you can use over and over again, rather than writing it out multiple times. Functions enable programmers to break down or decompose a problem into smaller chunks, each of which performs a particular task. This is helpful in the way that if we were to hard code our code we'd firstly have to find where each one is, change each one individualy, which just takes way to long. Now with functions you can do whatever whenever.
